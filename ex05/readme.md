# ex05 - Soma dos Dígitos

Este exercício lê um número inteiro fornecido pelo usuário e calcula a soma de todos os seus dígitos.

## Descrição

O programa solicita ao usuário que digite um número inteiro, separa cada dígito desse número e soma todos eles, exibindo o resultado final na tela.

## Como executar

1. Compile o código:
   ```sh
   gcc soma_digitos.c -o <nome_desejado>
   ```

2. Execute o programa:
   ```sh
   ./<nome_desejado>
   ```

3. Siga as instruções para inserir o número inteiro.

## Exemplo de uso

```
Digite um número inteiro: 2514
Soma dos dígitos: 12
```
